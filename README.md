# ServletCRUD

A **Java Servlet-based CRUD Web Application** using JSP, JDBC, and MySQL. This project demonstrates a simple web application where users can **register, log in, view records, update their details, and delete users** from a MySQL database. The project follows **MVC architecture** and includes a clean, user-friendly interface.

---

## **Features**

- **User Registration**: New users can sign up with name, username, email, contact, and password.  
- **User Login**: Authenticate users before allowing access to dashboard.  
- **View Data**: View all registered users in a tabular format.  
- **Edit & Update**: Update user information easily.  
- **Delete Users**: Remove users from the database.  
- **Clean UI/UX**: Simple, responsive, and user-friendly interface.  
- **Database Integration**: Uses MySQL with JDBC for backend connectivity.  

---

## **Technologies Used**

- **Java Servlets & JSP**  
- **JDBC** for database connectivity  
- **MySQL** database  
- **HTML/CSS** for UI  
- **MVC Architecture** for code organization
